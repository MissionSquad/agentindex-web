# AgentIndex Frontend

Astro + Vue + Vuetify frontend for AgentIndex routes:

- `/`
- `/agents`
- `/agents/[agentId]`
- `/reputation`
- `/reputation/[agentId]`
- `/address/[address]`
- `/tx/[txHash]`
- `/analytics`
- `/network`
- `/search`

## Env

- `PUBLIC_SCANNER_API_BASE_URL` (default: `http://localhost:3000`)

## Scripts

- `npm run dev`
- `npm run check`
- `npm run test`
- `npm run build`
